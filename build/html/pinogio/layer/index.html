레이어 개요

레이어 레이어는 업로드 및 편집된 데이터셋의 컬럼 값을 활용하여 데이터셋을 직관적으로 볼 수 있게 스타일을 변경합니다. 또한 프로젝트에 가입되어 있지 않은 외부 사람에게도 공유가 가능합니다.
